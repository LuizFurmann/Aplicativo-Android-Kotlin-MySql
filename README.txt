Para o funcionamento correto do aplicativo o usuário deve ter um banco de dados para que o app
possa ler, deixei o que usei para desenvolver.
usei o xampp como servidor, os arquivos php devem ficar na pasta htdocs, em uma pasta 
com o nome de sua preferência.
Para remover uma observação do app, pode colocar "null" no campo de observação para que
ela seja salva como nula e desapareça do app, fica como evolução melhorar isso hehe.
